# PracticeFirestoreCrud

# 🔥 Firestore CRUD Demo with Angular

This is a completed angular project created using practice-Firestore-CRUD-starter project to demonstrate **CRUD (Create, Read, Update, Delete)** operations using **Firebase Firestore**.

It uses Firestore as a NoSQL database to manage a `users` collection, allowing you to:

- Add a new user
- View all users
- Update user details
- Delete a user

## 🚀 Tech Stack

- **Angular**
- **Firebase Firestore**
- **Signals** for reactive state management

## 📦 Features

- 🔄 Real-time updates from Firestore
- 🔥 Seamless Firebase integration via Firebase SDK

### 🔥 Fork and Clone and FirebaseCongfig steps

1.  Fork and Clone the project on your local machine.
2.  Run `npm install` to install all the dependencies.
3.  Go to Firebase Console in **firebase.google.com**
    1. Create a new project
    2. Enable Cloud Firestore (in test mode)
    3. Copy the **firebaseConfig** object from your app's settings in Firebase project.
    4. Paste the firebaseconfig inside **firebase.config.ts** file.
4. Run the `ng serve` command to run the project.
